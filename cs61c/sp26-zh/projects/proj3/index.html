项目3：CS61CPU

A部分截止日期：2026年3月20日，星期五，晚上11:59:59（太平洋时间）

B部分截止日期：2026年4月9日，星期四，晚上11:59:59（太平洋时间）

在此项目中，您将构建一个运行实际RISC-V指令的CPU。

设置

您必须在本地机器上完成此实验。如果需要重新设置本地机器，请参阅实验0。

对于Logisim的常见错误或项目3特有的错误，请查看常见错误页面。

设置：Git

此作业可以单独完成或与合作伙伴一起完成。

警告：一旦您在Gradar上创建了小组，您将无法更改（添加、删除或交换）此项目的合作伙伴（项目3A和3B都是），因此请在开始项目之前确认您的合作伙伴。您必须在Gradar上和每次Gradescope提交时添加您的合作伙伴。

如果有特殊情况需要更换合作伙伴（例如您的合作伙伴退课、您的合作伙伴没有回应），请私下联系我们。

  1. 访问Gradar。登录并注册您的项目3小组（如果有合作伙伴，请添加您的合作伙伴），然后为您或您的小组创建一个GitHub仓库。如果您有合作伙伴，一位合作伙伴应创建小组并邀请另一位合作伙伴加入该仓库。另一位合作伙伴应接受邀请，而不是创建自己的小组。

  2. 克隆仓库到您的工作空间。请使用您的本地机器（此项目完全不需要hive机器）。Windows用户应在WSL外部克隆（建议使用Git Bash）。

    git clone git@github.com:61c-student/sp26-proj3-USERNAME.git 61c-proj3

    （替换USERNAME为您的GitHub用户名）

  3. 导航到您的仓库：

    cd 61c-proj3

  4. 添加起始仓库作为远程：

    git remote add starter https://github.com/61c-teach/sp26-proj3-starter.git

如果您遇到git问题，请查看常见错误页面。

设置：Logisim

此项目在Logisim中完成。在61c-proj3目录中，运行bash test.sh download_tools以下载此项目的Venus和Logisim。（您只需运行一次。）

在项目的其余部分中，要打开Logisim，运行java -jar tools/logisim-evolution.jar。

恢复起始文件

要恢复起始文件，请查看常见错误页面。
