Quest考试

Quest详情

Quest是一个50分钟的PrairieLearn风格作业，需在计算机化考试中心(Computer-Based Testing Facility, CBTF)面对面完成。您可以通过PrairieTest直接在任何可用时间段预约1小时的考试时间。如果您有DSP特殊安排，请直接通过AIM门户提交您的安排信，签到时将自动获得特殊安排。

CBTF流程

CBTF位于Soda 275和Sutardja Dai 200。请务必查看RTL的CBTF页面了解政策和程序，并查看"学生入门"页面了解您需要完成的任务。

请仔细阅读所有设施流程！

待办事项

  • 您必须登录PrairieTest来创建您的PrairieTest账户。
  • 尽快提交任何DSP特殊安排，以便输入到您的PrairieTest账户中。未能创建账户将导致CBTF工作人员无法输入您的特殊安排。
  • 当自主预约开放时，为您的考试进行预约。预约说明请见这里。
  • 下载并使用您的CalNet账户设置Duo Mobile。Soda 275的计算机将要求您在访问考试前使用Duo进行身份验证。CBTF内短信可能不稳定，不推荐作为双因素认证方式。认证问题不会获得额外时间。
  • 如果您错过预约，请发送邮件至cbt-help@berkeley.edu寻求帮助。

一般提醒

  • 考试期间不允许使用任何未授权设备（智能手机、个人作弊纸、智能手表等）。
  • 监考人员将监控CBTF，并将任何学术不端行为直接报告给课程工作人员。
  • CBTF监考人员在考试期间不能回答任何与内容相关的问题。
  • 请确保在预定时间前10分钟到达，以便准时签到。考试将在列出的时间准时开始。

如果您对与您情况相关的流程有任何疑问，请发送邮件至cs61c@berkeley.edu。

定向测验

在2月12日星期四和2月13日星期五有一个不计分的定向测验，旨在让您熟悉设施。这是可选的，但建议参加，以便在正式参加Quest之前了解政策和场地。定向测验时长20分钟，可以在PrairieTest上自主预约。

评分标准

Quest考试总计25分：

  • 数字表示（共4分）
    • 进制转换（2分）
    • 有符号/无符号转换（2分）
  • C语言（共17分）
    • IEC（1分）
    • C代码求值（12分）
    • 位运算/位操作（4分）
  • 浮点数（共4分）
    • 机械计算（2分）
    • 概念理解（2分）

重考

重考安排在Quest考试后的一周。所有学生都有资格参加重考，您的Quest成绩将是Quest和重考成绩的最高分(superscore)。例如，下表展示了一个计分计算示例。注意：示例评分细分可能不代表Quest的实际细分。它仅作为计算成绩的示例。

Quest Retake Superscore
Num Rep (Bases) 1 / 2 2 / 2 2 / 2
Num Rep (Signed / Unsigned) 2 / 2 0 / 2 2 / 2
C (IEC) 1 / 1 0 / 1 1 / 1
C (Code) 9 / 12 11 / 12 11 / 12
C (Bitwise) 3 / 4 4 / 4 4 / 4
Floating Point (Mechanical) 1 / 2 1 / 2 1 / 2
Floating Point (Conceptual) 0 / 2 2 / 2 2 / 2
Overall 17 / 25 20 / 25 23 / 25

由于重考为所有学生提供了第二次参加考试的机会，在指定时间之外将不会有其他重考或替代考试安排。请注意，由于最高分政策，跳过Quest或重考不会有处罚，但如果您无法参加两次考试，您将在Quest部分获得零分。此政策无例外，因为已有6天可以参加Quest/重考。
